### 🚀 Flagship Project: [Ozigi.app](https://github.com/Dumebii/OziGi)
Ozigi is a universal Context Engine built to cure "blank page syndrome" for technical creators.
* **The Problem:** LLMs generate robotic, highly-predictable copy that requires heavy prompt engineering.
* **The Solution:** I engineered a Next.js / Vertex AI pipeline that ingests raw, unformatted context (PDFs, URLs, meeting notes) and maps it to a database-backed voice persona.
* **The Architecture:** It utilizes strict programmatic constraints (like a "Banned Lexicon" to block words like *delve* and *tapestry*) to bypass AI detection and output bursty, pragmatic, multi-platform campaigns.
